# EndPortalPlugin 🔮

Plugin para o servidor **NewsCraft** que permite coletar e recolocar
blocos do Portal do End usando uma picareta com Toque Suave (Silk Touch).

## Requisitos

- Paper/Spigot 1.21.x
- Java 21

## Como usar

1. Tenha a permissão `newscraft.endportal`
2. Pegue uma picareta com **Toque Suave**
3. Quebre um bloco do Portal do End
4. O bloco vai direto para o seu inventário
5. Coloque o bloco onde quiser

## Permissões

| Permissão | Descrição | Padrão |
|---|---|---|
| `newscraft.endportal` | Permite coletar e recolocar blocos do Portal do End | OP |

## Compilação

O plugin é compilado automaticamente pelo GitHub Actions
a cada push na branch `main`. Baixe o `.jar` em:
**Actions → Build EndPortalPlugin → Artifacts**
